# StickerShop

A web application or digital assets store where users (sticker lovers) can get any sort of WhatsApp stickers for their chats at a very low price (NGN 300).

## Technologies used
HTML, CSS, JavaScript, Paystack integration (for payments) and Cocobase (a BaaS for my back-end server/system and DB).
